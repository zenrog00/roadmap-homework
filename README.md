## Roadmap homework

### Description
Pet project with creating web app in NestJS.

### Stack
* NodeJS
* Typescript
* NestJS
* PostgreSQL
* Docker
* TypeOrm
* Passport
* Swagger

### Setup
To setup this project you need to create .env file in project root with contents of .env.example file and fill it with your configuration.

Then install npm packages with `npm ci`.

### Usage

To run PostgreSQL database use `docker compose up -d`.

To run project use `npm run start:dev`.

To run tests use `npm run test:e2e`.
